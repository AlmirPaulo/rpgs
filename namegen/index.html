Gerador de Nomes de Fantasia

Ajuda

Símbolos

  • s sílaba genérica
  • v vogal
  • V vogal ou combinação de vogais
  • c consoante
  • B combinação de consoantes compatível com o começo de uma palavra
  • C combinação de consoante compatível com qualquer parte de uma palavra
  • i insulto (em inglês)
  • m nome piegas (em inglês)
  • M nome com final piegas
  • D consoante adequada para um nome idiota de uma pessoa
  • d sílaba adequada para um nome idiota de uma pessoa (começa com uma vogal)

Sintaxe

  • <> Os símbolos entre < > geram fragmentos (veja acima).
  • () O conteúdo entre parênteses aparece literalmente.
  • | O gerador selecionará a esquerda ou a direita da barra..
  • ! O componente seguinte aparece em maiúsculo.
  • ~ Inverte o componente seguinte.
